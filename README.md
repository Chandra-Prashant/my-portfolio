## 🛠 Technical Architecture

### **Core AI Stack**
* **Frameworks:** PyTorch, Scikit-learn, OpenCV
* **Algorithms:** ResNet-50 (CNN), MoveNet (Pose Estimation), LBPH (Face Rec)
* **Optimization:** Grad-CAM for visualization, Vector Geometry for tracking.

### **Engineering Stack**
* **Framework:** Next.js 15 (App Router) + TypeScript
* **Styling:** Tailwind CSS (Scientific Dark Theme)
* **Visuals:** HTML5 Canvas (Mathematical Matrix/Binary Rain)

## 💻 Local Development (Academic Lab Setup)

1. **Clone the Environment:**
   ```git clone [https://github.com/Chandra-Prashant/my-portfolio.git](https://github.com/Chandra-Prashant/my-portfolio.git)```

2. **Install Dependencies:**
    ```
    cd my-portfolio
    npm install
    ```
3. **Initalize Research Server**
   ```
    npm run dev
   ```
4. **Access:**
   Navigate to http://localhost:3000 to view the live build.
